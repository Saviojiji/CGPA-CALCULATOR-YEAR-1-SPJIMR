# 🎓 SPJIMR CGPA Calculator — PGDM 2025-27

A real-time CGPA calculator built for **SPJIMR PGDM 2025-27** students to track Term GPAs, cumulative CGPA, and promotion eligibility.

## 🌐 Live Demo

### 👉 [**Open the Calculator**](https://saviojiji.github.io/CGPA-CALCULATOR-YEAR-1-SPJIMR/)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📚 **Pre-loaded Curriculum** | All courses for Term 1 (16 cr), Term 2 (18 cr), Term 3 (15 cr), and NCL (5 cr) |
| 📝 **Course-wise Grade Entry** | Select grades (A+ to F) for each course individually |
| 🔄 **Direct GPA Input Mode** | Toggle to enter Term GPA directly instead of per-course grades |
| 📊 **Combined GPA Calculator** | See GPA for any combination — T1+T2, T1+T3, T2+T3, T1+T2+T3, and more |
| ✅ **Promotion Rules Engine** | Auto-checks Rule 2.10 — Term 1 ≥ 2.4, Comp Exam clearance, max F/D limits |
| 🎯 **NCL Courses** | Abhyudaya, ADMAP (absolute grading), Comprehensive Examination |
| ⚡ **Real-time Updates** | GPA recalculates instantly as you change any input |

## 🏗️ Run Locally

```bash
# Clone the repo
git clone https://github.com/Saviojiji/CGPA-CALCULATOR-YEAR-1-SPJIMR.git

# Navigate into the project
cd CGPA-CALCULATOR-YEAR-1-SPJIMR

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open **http://localhost:5173/CGPA-CALCULATOR-YEAR-1-SPJIMR/** in your browser.

## 📋 Promotion Rules (Rule 2.10)

- ✅ Term 1 GPA ≥ 2.4
- ✅ Comprehensive Exam — No F grade
- ✅ Cumulative GPA ≥ 2.4 (after Term III)
- ✅ Maximum 1 F grade overall
- ✅ Maximum 3 Ds (if no F) or 1 D (if has F)
